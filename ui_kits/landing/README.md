# ClarityWorks Landing Page — UI Kit

A high-fidelity recreation of the ClarityWorks marketing landing page,
factored into reusable JSX components.

## Files

- `index.html` — the assembled page (this is what renders)
- `Primitives.jsx` — `Button`, `Card`, `Section`, `Eyebrow`, `Container`, `Icon`
- `Navbar.jsx` — sticky top nav, tinted on scroll
- `Hero.jsx` — display headline + lede + CTA pair + proof strip
- `Services.jsx` — three core services
- `ProofMetrics.jsx` — four-up metric band
- `EngagementProcess.jsx` — seven-step process flow
- `CaseStudies.jsx` — three case study cards
- `Differentiators.jsx` — five-point manifesto-style list
- `ContactCTA.jsx` — book a discovery call form
- `Footer.jsx` — navy footer, clarityworks.com.au, ClaritySRM mention

## How to run
Open `index.html`. All components load via inline JSX + Babel.

## What this kit does NOT do
- Real form submission — the contact form is cosmetic.
- Mobile breakpoints below 720px are sketched but not polished.
- No real client logos — placeholder marks only.
